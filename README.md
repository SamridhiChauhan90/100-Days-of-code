# 100 Days of Code

## Day 1: Sum of Two Numbers in C
- **Goal:** Learn to take input and output in C
- **Code:** day1.c
- **Description:** Program takes two integers from the user and prints their sum.
- **Key Learning:**
  - `scanf` & `printf`
  - Variables & addition
  - Basic program structure (`#include <stdio.h>` and `main()` function)
